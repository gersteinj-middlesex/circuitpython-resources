For Loop


							for n in range(10):
								print(n)

This is even shorter! Instead of having to create a variable and then update it in the loop, the for loop uses range() to generate a set of numbers, then automatically assigns each value to the variable

Range Variations

Using The Range Function

The range() function is actually something called a generator. It doesn't give you all the specified numbers directly, but you can loop through it to access the numbers

Maximum only


							range(10)

This version generates a set of numbers going from 0 to 10, not including the maximum

Minimum and Maximum


							range(3, 10)

This version generates a set of numbers going from 3 to 10, not including the maximum

Minimum, Maximum, Step


							range(20, 3, -2)

This version generates a set of numbers going from 20 to 3, changing by -2 each time (so subtracting 2, instead of adding), not including the endpoint (so 3 isn't included)

Fading from Red To Magenta

  • red is (255, 0, 0) and magenta is (255, 0, 255)
  • to get from red to magenta, you need to think about what each component of the color does
  • (?, ?, ?)
  • (255, ?, ?)
  • (255, 0, ?)
  • (255, 0, increase)
  • Only the blue component changes, and we just looked at how to change a single value!

Code


							# Assume the code to set up the built-in RGB LED exists above here

							for n in range(256):
								led[0] = (255, 0, n)

Fading from Magenta to Blue

  • magenta is (255, 0, 255) and blue is (0, 0, 255)
  • Once again, think about what each component of the color does
  • (?, ?, ?)
  • (decrease, ?, ?)
  • (decrease, 0, ?)
  • (decrease, 0, 255)
  • Only the red component changes. But now we need to decrease it! Can we change the range() function to do that?

Code


							# Assume the code to set up the built-in RGB LED exists above here

							for n in range(255, -1, -1):
								led[0] = (n, 0, 255)

Fading from Yellow to Magenta

  • yellow is (255, 255, 0) and magenta is (255, 0, 255)
  • Once again, think about what each component of the color does
  • (?, ?, ?)
  • (255, ?, ?)
  • (255, decrease, ?)
  • (255, decrease, increase)
  • Now we need one value to increase and one to decrease! They're both in the same range, though...

A little bit of math

Don't worry, the math itself is easy

  • We have two numbers that should be going in opposite directions, and a variable that's going from 0 -> 255
  • The number that's going backwards should be 0 when the variable is 255, and should be 255 when the variable is 0
  • ? - n
  • what does the question mark represent

Code


							# Assume the code to set up the built-in RGB LED exists above here

							for n in range(256):
								led[0] = (255, 255 - n, n)
