    Stepping Between Values

    Often, you'll need your code to be able to go smoothly from one value to another. A few examples: - When you're in a car, you accelerate from stopped to going full speed, and brake gradually when you come to a stop - With LEDs, you might want to blend smoothly from one color to another - If you had a robotic arm, you'd want to be able to have positions between fully closed and fully opened.
